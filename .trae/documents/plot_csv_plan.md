# Python 数据可视化与滤波效果分析脚本计划

## 1. 摘要 (Summary)
在 `scripts` 文件夹下新增一个数据绘图脚本 `plot_csv.py`。该脚本利用 `matplotlib` 对之前记录的 CSV 电机数据进行可视化，并通过交互式的终端输入来选择数据文件、指定需要截取的起始/结束行，以及需要对比的列号。此外，脚本会计算并输出数据的“标准差(Std Dev)”和“差分标准差(Std Diff)”，帮助用户直观量化低通滤波器的平滑效果。

## 2. 现状分析 (Current State Analysis)
- `rx_csv.py` 生成的 CSV 数据文件格式为纯文本，每行 8 个逗号分隔的数值，分别对应 4 个电机的原始 RPM 和滤波后 RPM。
- 当前缺少直观的波形查看工具，也不具备衡量滤波好坏（如高频噪声被滤除的程度）的客观指标。
- 用户倾向于交互式的 CLI 体验，以及使用“标准差”与“差分标准差”作为评价滤波平滑度的指标。
- 需要用户通过 `uv` 额外安装数据处理和绘图库（`numpy`, `matplotlib`）。

## 3. 拟进行的更改 (Proposed Changes)

### 3.1 编写绘图与分析脚本 `scripts/plot_csv.py`
- **依赖库**: `numpy`, `matplotlib`
- **文件扫描与选择**:
  - 启动时自动搜索当前运行目录及 `scripts` 目录下的所有 `.csv` 文件，列出带序号的菜单供用户选择，避免手动输入长文件名。
- **交互式参数输入**:
  - 提示用户输入起始行号 (Start Row) 和结束行号 (End Row) 以截取感兴趣的波形段。
  - 提示用户输入需要绘制的列号（例如输入 `0,1` 代表同时绘制第一列和第二列，即电机1的原始值和滤波值）。
- **数据加载与截取**:
  - 使用 `numpy.loadtxt` 或 `numpy.genfromtxt` 读取 CSV 数据，提取指定的行列范围。
- **量化指标计算**:
  - 对用户选中的每一列数据，计算以下指标：
    1. **均值 (Mean)**：这段时间的平均转速。
    2. **标准差 (Std Dev)**：反映整体波动范围。对于恒速稳态过程，该值越小说明波动越小。
    3. **差分标准差 (Std of Diff)**：即相邻两点之差的标准差 (`np.std(np.diff(data))`)。这是衡量曲线平滑度的极佳指标，低通滤波器应当能显著降低该值。
- **可视化绘制 (Matplotlib)**:
  - 将选中的列绘制在同一个图表中，加上网格线 (Grid)、图例 (Legend) 和标题。
  - 在终端中打印出上述计算的量化指标对比表格，方便直观查看滤波前后的数据改善情况。

## 4. 假设与决策 (Assumptions & Decisions)
- **数据结构假设**: 假设 CSV 中没有中文字符表头（因为目前 `rx_csv.py` 中的表头写入是注释掉的），如果遇到解析错误会自动跳过或提示。
- **轻量化处理**: 使用 `numpy` 代替 `pandas`，因为单纯的矩阵切片和基础统计计算使用 `numpy` 足以胜任，减少依赖体积。
- **列索引基准**: 交互提示中，列号从 0 开始（0 代表第一列，1 代表第二列...），会在终端提示语中做清晰说明。

## 5. 验证步骤 (Verification Steps)
1. 检查 `scripts/plot_csv.py` 是否成功生成。
2. （用户操作）执行 `uv pip install numpy matplotlib` 安装所需依赖。
3. （用户操作）运行 `python scripts/plot_csv.py`。
4. （用户操作）根据终端提示选择之前抓取的 `data_*.csv` 文件，输入截取范围（如 `0` 到 `1000`），并输入列号 `0,1`。
5. 验证终端是否正确输出了两列数据的“均值”、“标准差”和“差分标准差”。
6. 验证是否弹出了包含两条曲线对比的 Matplotlib 绘图窗口。
